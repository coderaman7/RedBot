### Installation

1. Get your Reddit API Key at [https://www.reddit.com/prefs/apps/](https://www.reddit.com/prefs/apps/)
2. Clone the repo
   ```sh
   git clone https://github.com/coderaman7/redbot.git
   ```
3. Install NPM packages
   ```sh
   pip install -r requirements.txt
   ```
